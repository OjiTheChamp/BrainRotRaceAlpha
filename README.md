# 🧠 Brainrot Speedrun

A chaotic browser racing game where you train your brainrot character on a treadmill before competing in a high-speed race against AI rivals.

---

## 🎮 How to Play

1. **Pick your character** from the title screen
2. **Train on the treadmill** — mash ← → keys to build up speed before the race starts
3. **Race!** — reach the finish line before the 13-minute timer runs out

---

## ✨ Features

- **6 playable brainrot characters** — Skibidi, Sigma, Rizz Lord, Gyatt, Fanum, and NPC — each with unique speed and boost stats
- **Treadmill training phase** before every race — the harder you mash, the faster you start
- **All racers begin at 0 speed** and accelerate from scratch
- **Power-ups** scattered across the track:
  - ⚡ Rizz Boost — refills your boost meter
  - 🚀 No Cap Turbo — supercharges your speed for 11 seconds
  - 🛡️ Sigma Shield — makes you immune to obstacles
  - 🍕 Fanum Tax — slows down all rivals
- **Obstacles** to dodge across 5 lanes
- **13-minute race timer** — finish before time runs out or you lose
- **Tetris Theme A** synthesized live with the Web Audio API
- **Full sound effects** — engine rumble, boost fire, crashes, power-up jingles, countdown ticks, win/lose fanfares
- **No dependencies, no installs** — single `.html` file, runs entirely in the browser
- **Mobile touch controls** included

---

## 🕹️ Controls

| Action | Keyboard | Mobile |
|---|---|---|
| Accelerate | `↑` / `W` | ▲ button |
| Steer left | `←` / `A` | ◀ button |
| Steer right | `→` / `D` | ▶ button |
| Boost | `Space` / `Shift` | ⚡ BOOST button |
| Treadmill mash | `← →` / `Space` | TAP button |
